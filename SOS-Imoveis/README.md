## 📝 Descrição do projeto

Aplicação mobile em React Native + Expo + TypeScript para gerenciar corretores, clientes e imóveis com armazenamento local em SQLite.

## 🤔 Problema Social

Muitas pessoas enfrentam dificuldades para encontrar imóveis adequados devido à falta de organização das informações e à comunicação ineficiente entre clientes e corretores. O projeto busca facilitar esse processo por meio de uma plataforma digital que centraliza o gerenciamento de imóveis, clientes e visitas.

## 🧰 Funcionalidades principais

- Cadastro e edição de corretores, clientes e imóveis
- Login e cadastro de usuários
- Favoritos e avaliações de imóveis
- Navegação entre telas
- Persistência local com Expo SQLite

## 📦 Tecnologias utilizadas

- React Native
- Expo SDK 54.0.0
- TypeScript
- Expo SQLite
- React Navigation
